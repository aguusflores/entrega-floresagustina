NUESTRAS CABAÑAS

Están cuidadosamente diseñadas para brindarte comodidad y lujo en medio de la belleza natural de las montañas. Cada cabaña cuenta con:
  • Vistas panorámicas de las majestuosas montañas desde todas las cabañas.
  • Cocinas completamente equipadas con electrodomésticos modernos.
  • Sala de estar acogedora con chimenea para las noches frías.
  • Amplias terrazas con parrillas para disfrutar de comidas al aire libre.

CARACTERISTICAS DEL RESORT

  • Senderos para caminatas con acceso directo desde el complejo.
  • Área de fogata para reuniones nocturnas bajo las estrellas.
  • Piscina climatizada con vistas a las montañas.
  • Zona de juegos para niños.
  • Wi-Fi gratuito en todo el complejo.
